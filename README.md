# ROOT XTRONS QTB8060CCC
Starting off my 1st 2026 Github post with a personal project of mine. Obtaining root on the XTRONS QTB8060CCC Android headunit which was made for the E6X and E9X BMW Platform using the CCC i-Drive system

Requirements:
8GB+ USB flash drive,
Laptop ideally unless you have a bench power supply,
USB A to A. You can also use a USB C to A adapter
